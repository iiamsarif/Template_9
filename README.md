## ğŸ¨ Design Specifications

### Color Palette
- Primary Red: `#C4001A` (Premium accent color)
- White: `#FFFFFF` (Clean background)
- Soft Gray: `#F5F5F5` (Section backgrounds)
- Dark Text: `#111111` (Primary text color)

### Typography
- Font Family: Inter (Google Fonts)
- Weights: 200 (Light), 300 (Regular), 400 (Medium), 500 (Semi-bold), 600 (Bold), 700 (Extra-bold), 800 (Black)
- Headings: Bold weights with tight line-height
- Body Text: Light weights with generous line-height

### Breakpoints
- Desktop: 1024px and up
- Tablet: 768px - 1023px
- Mobile: 767px and below

## ğŸš€ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)

1. Clone or download the project

- **Colors**: Modify CSS variables in `style.css`

- **Animations**: Adjust GSAP timelines in `main.js`
- **Contact**: Update form submission endpoint in JavaScript
